# JPMorgan Chase Quantitative Research Job Simulation

This repository contains my solutions for the JPMorgan Chase & Co. Quantitative Research Virtual Experience Program on Forage.

## Tasks Completed

### Task 1: Natural Gas Price Analysis
- Loaded and analyzed historical natural gas price data.
- Visualized price trends using Python and Matplotlib.
- Estimated future gas prices based on historical patterns.

### Task 2: Commodity Storage Contract Pricing
- Built a pricing model for a natural gas storage contract.
- Calculated contract value using:
  - Injection price
  - Withdrawal price
  - Storage costs
  - Volume stored

### Task 3: Credit Risk Analysis
- Trained a Logistic Regression model to predict Probability of Default (PD).
- Evaluated model performance using accuracy metrics.
- Calculated Expected Loss (EL) using predicted default probability.

### Task 4: FICO Score Bucketing
- Created a rating system for customer credit scores.
- Applied quantization techniques to group FICO scores into risk buckets.
- Generated bucket boundaries for credit risk assessment.
